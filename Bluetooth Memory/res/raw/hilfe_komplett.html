Bluetooth Memory

1. Was ist BlueMemory?

BlueMemory ist eine einfache Implementierung des bekannten Spiels "Memory", das sich mit mehreren Spielern über eine Bluetooth-Verbindung spielen lässt. Das Spiel enstand im Rahmen des Praktikums "Android-Anwendungsentwicklung" an der Technischen Hochschule Mittelhessen im Wintersemester 2010 / 2011.

2. Das Spiel

2.1 Spiel erstellen

Um ein Spiel zu eröffnen, klicken Sie auf "Spiel erstellen". Im folgenden Bild können Sie die Größe des Spielfelds (Höhe x Breite), das Kartendeck und die Schwierigkeitsstufe auswählen. Die Auswahl Schwierigkeitsstufe beeinflusst die Pause, die vom Spiel zwischen zwei Zügen erzwungen wird. Bitte beachten Sie, dass die Pause auch von der Rechenleistung der am Spiel beteiligten Geräte und der Verbindungsqualität abhängig sein kann. Falls die Geräte der anderen Spieler noch nicht mit Ihrem verbunden sind, muss die Bluetooth-Sichtbarkeit vorher über die Einstellungen Ihres Gerätes aktiviert werden.

Klicken Sie auf "Spiel erstellen", um zur "Lobby" zu wechseln.

2.2 Spiel teilnehmen

Um an einem Spiel teilezunehmen, klicken Sie auf "Spiel teilnehmen". Der Bluetooth-Adapter wird dann aktiviert und es erscheint eine Liste der verfügbaren Bluetooth-Geräte in Ihrer Umgebung. Geräte, mit denen Ihr eigenes Gerät bereits verbunden ist, werden sofort angezeigt, gleichzeitig wird nach weiteren Geräten gesucht. Neu gefundene Geräte werden der Liste nach und nach hinzugefügt. Wählen Sie eines der Geräte, um einem Spiel beizutreten und zur Lobby zu gelangen. Befindet sich auf dem gewählten Gerät kein BlueMemory-Server in Betrieb, wird die Lobby nach kurzer Zeit automatisch wieder verlassen.

2.3 Die Lobby

In der Lobby befinden sich alle Spieler, die an einem Spiel teilnehmen wollen. Es können maximal 6 Spieler an einem Spiel teilnehmen. Die Option zum Starten des Spiels steht dabei nur dem Server zur Verfügung. Klicken Sie auf "Spiel starten", wenn Sie das Server-Gerät bedienen, um das Spiel zu beginnen. Nehmen Sie an einem Spiel teil, warten Sie, bis der Server das Spiel startet.
